# eReader Leaks
## Exlibris generator using Calibre database, of some books that were in my eReader.



## REQUIREMENTS

 - Python 3.x

## USAGE

- Generate HTML libraries:
```
$ python librero.py
$ python improve2.py

```
- Edit the JSON:
```
$ python editor.py

```
- Edit the JSON via HTML interface:
```
$ open ./editor.html

```
